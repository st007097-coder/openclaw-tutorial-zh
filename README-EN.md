## What is OpenClaw?

OpenClaw is an open-source AI assistant framework that connects to your everyday messaging platforms (Telegram, WhatsApp, Discord), automates your workflows, and keeps your data private.

**Key Features:**
- 🔗 **Multi-platform** — Telegram / WhatsApp / Discord / iMessage / Signal / Slack
- 🧠 **Long-term Memory** — Remembers what you teach it across sessions
- 🛠️ **Tool Integration** — Run commands, browse the web, manage files
- ⏰ **Automation** — Heartbeat, Cron Jobs, Webhooks
- 🔒 **Privacy First** — Self-hosted, your data stays yours
- 📱 **Mobile Nodes** — Camera, GPS, voice from your phone

## Quick Start

```bash
# Install
npm install -g openclaw

# Start the gateway
openclaw gateway start

# Connect Telegram
openclaw channel add telegram
```
